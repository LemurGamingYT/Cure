# Cure
 A programming language compiled to C

**Features**:
---
- **Speed**: Cure is compiled to C, and therefore machine code, and so it will be fast.
- **Static typing**: Cure is statically typed, increasing type safety and performance. Cure does have type inference for variables and function return types (except for recursive functions).
- **C API**: Since Cure is written in C, you can use Cure's types and functions in C too (check examples/c for some examples on the C API)
- **Simplicity**: Cure has a simple syntax almost like a bridge between Python and C.
